# chip8-emu
A chip8 emulator made in C++ with OpenGL, currently it is windows only.
## Usage
`chip8 ROM_file [--log [--log-regs]]`<br/>
with `--log`, CPU instructions are printed in the console<br/>
with `--log-regs`, registers are printed too<br/>
